galapagos

A simple and cross-plataform graphical tool to visualize evolution on genetic programming algorithms

Install

You can either grab one binary release or:

  1. Download the Qt SDK
  2. Run qmake on the base folder
  3. Compile using make (on Linux), XCode (on Mac OS X) or Visual Studio (on Windows)
  4. Run

License
